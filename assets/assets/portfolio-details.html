Sistem Analisis Banduan Antarabangsa dan Malaysia

This project is a comprehensive prisoner data analytics system designed to analyze and manage information across both international and local contexts. It provides insights, visual reports, and a database management interface to assist authorized departments in data-driven decision-making.

Key Features

Integrated Database

Stores and processes prisoner data securely using MySQL with optimized queries.

Data Visualization

Includes analytical graphs and statistical dashboards built with Power BI and Looker Studio.

Secure Access

Implements authentication and authorization controls for data protection.

Scalable Design

Developed using PHP and Laravel with modular and scalable architecture.
